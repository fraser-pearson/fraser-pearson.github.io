Problem Solving

What was the problem?

I was failing to get the SuperFizzBuzz program to work. It was meant to take a list of number and output fizz if it was divisible by 3, buzz if by 5 and fizzbuzz if by both.

What problem-solving techniques did you use?

The problem solving techniques I used were mainly console.logging and reading error messages. I used the console.I used the techniques to find where the issue was occurring and from there it was easy enough to fix the code.

How did you feel throughout the process?

I did not feel bad throughout the process. I never felt like I wouldnt be able to find a solution in a reasonable amount of time. I knew what I needed to do was just pushing through it now.

What did you learn?

I learned the importance of using console.log to see what your code outputs and see what different error messages mean.
